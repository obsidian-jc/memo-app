GitHub Personal Access Token을 1회 입력하면
메모가 Gist에 자동으로 저장·불러오기 돼요.
기기 바꿔도 동일한 메모를 볼 수 있어요.

토큰 발급 방법:

  1. github.com → Settings → Developer settings → Personal access tokens → Tokens(classic)
  2. Note: 아무 이름 입력 (예: memo-app)
  3. Expiration: No expiration 선택
  4. Select scopes: gist 체크 ✓
  5. Generate token → 복사

🏷 태그 활용법

태그는 두 가지 방식으로 붙여요.

자동
저장 시 키워드를 감지해서 자동으로 태그가 제안돼요. 백업 탭에서 키워드를 직접 편집할 수 있어요.
수동
메모 탭에서 태그 버튼을 클릭하거나 + 직접입력으로 커스텀 태그를 달아요.

💡 제텔카스텐에서는 태그를 주제어로 쓰세요. 예: #관리철학 #누수대응

📅 달력 · 할일

PC에서는 오른쪽 상단에 달력과 할일이 항상 보여요. 모바일에서는 메모 탭 하단에 오늘 할일 위젯이 있어요.

1
달력에서 날짜 탭 → 그날 할일 목록 표시
2
할일 입력 후 Enter 또는 ＋ 버튼으로 추가
3
할일 탭 → ✓ 완료/미완료 토글 · ✕ 삭제

💡 할일 있는 날은 달력에 ● 점으로 표시돼요.

⚠️ 할일은 그 기기에만 저장돼요. Gist 동기화와는 별개예요.

1
목록에서 카드 펼치기 → ✏️ 수정 버튼
2
수정 후 ✓ 저장 → 카드에 수정됨 배지 표시

원본은 내부적으로 보존돼요.

⚠️ 중요 주의사항
토큰은 절대 채팅창·메모·URL에 붙여넣지 마세요!
토큰이 노출되면 즉시 GitHub에서 삭제하고 새로 발급하세요.
1
github.com/settings/tokens/new 에서 발급
2
Expiration: No expiration · Scope: gist 만 체크
3
각 기기마다 한 번씩 백업 탭에서 직접 입력
4
토큰은 메모장 등 안전한 곳에 따로 보관하세요
기기별 동기화 순서
1. 메모 많은 기기에서 먼저 ⬆ 업로드
2. 다른 기기에서 토큰 입력 → 연결 저장
3. 공유 URL 복사 → 다른 기기 브라우저에서 열기
4. ⬇ Gist에서 불러오기 → 병합

💡 공유 URL에는 토큰이 포함되지 않아요. Gist ID만 포함돼서 안전해요.
